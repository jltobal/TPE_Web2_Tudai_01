# TPE_Web2_Tudai_01

Trabajo practico especial para la materia de WEB 2 en la carrera de la Tecnicatura Universitaria en Desarollo de Aplicaciones Informaticas de la Facultad de Ciencias Exactas, sede Olavarría.

Puntos a completar:
*Acceso publico a la pagina
*Contenido generado dinamicamente desde una base de datos
*Visualizacion de un listado de items, detalle de item, listado de categoria y listado de item por categoria.
*Acceso de tipo Administrador a la base de datos.
*Logueo con usuario y contraseña.
*Logout.
*Creacion de servicio ABM para el administrado donde agregue items, edite items, elimine items, agregue categorias, edite categorias y elimine categorias.
*Utilizacion de patron MVC.
*Utilizacion de motor SMARTY.
*Utilizacion de URL semanticas.
*Incluir archivo SQL para instalar y probar la base de datos.
